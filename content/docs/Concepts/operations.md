Supported binary operations are:

```js {filename=variables.rash}
  x = 5; // Assignment
  x + 4; // Adittion
  x - 1; // Subtraction
  x * 4; x / 2; // Multiplication and division
  x == 4 && x <= 2; // And, or, equals, lesser, greater etc.
```
