Rash has a dynamic type system not to dissimilar to pythons, variables are declared like so:

```js {filename=variables.rash}
  name = 5; // Optional semi-colon
  print(name);

  text = "Hello World"

  rash_is_cool = true;

  arrr = [0, 4, 2, 1]
```
